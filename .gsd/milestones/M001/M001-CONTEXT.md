# M001 Context — Contest-Ready Web App

## Scope

Build the complete "Con Đường Điện Biên" web app: 5 gates with narrative storytelling, interactive puzzles, collectible character cards, badges, and hidden facts. Deliverables are a hosted web link and a video walkthrough.

## Goals

1. A working 5-gate interactive experience a judge can play through in a browser
2. Visually impressive blend of vintage war-map and modern editorial design
3. Clear AI usage documentation for the contest form

## Constraints

- Timeline: ASAP
- Tech: vanilla HTML/CSS/JS only, no frameworks, no build step
- Hosting: GitHub Pages
- Content: based on "Kể chuyện Điện Biên Phủ" (NXB Kim Đồng)
- Audience: contest judges + ages 13–17

## Key Decisions

- Blended visual style: vintage parchment/map + bold modern typography, anchored on the Điện Biên Phủ topographic map
- Single-page data-driven architecture
- Son acts as creative director, agent builds
